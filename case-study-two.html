Government Agency Redesign
And UI Redesign

Problems

This website needed a few minor changes. When looking at the homepage there is multiple categories and links which was a bit overwhelming. Being the user I didn't quite really understand what this website was about. I found the search button a bit too small to catch my eye. I've also noticed that the pictures used in the website were cut off to where you couldn't see it fully.

Our Solution

Simplifying the categories and links to a small amount but still having the major categories will be easier for the users to navigate through the homepage. Making the search bar more visible will be one of the most important tools. Changing the layout of the text and pictures to where you can see a full image will be important. Pictures help users to see more of the “mood” for the website and each paragraph. These are all very simple changes the will help the design.

Breaking Down Navigation

First focus was to simplify the navigations which was the first focus. Creating a graph of each individual category helped the starting of the redesign for the navigation. This help see what was necessary and what could have it's own section. For example shop was constantly was used in each L2.

Goals

  • Reduce Navigation
  • Create A Hamburger Menu
  • Show Clear Images
  • Users easlily recognize the main goal of Australia Post
  • Creating a simple footer that won't confuse users allowing them to easliy find support

Design Journey

  • Usability testing interview
  • Created a 2x2 matrix to understand the importance of the user and government agency. It may not be the same for the user and government. The government agency usually wants to know a lot of information of the users.
